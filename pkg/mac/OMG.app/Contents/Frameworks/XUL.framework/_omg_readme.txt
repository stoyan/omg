http://releases.mozilla.org/pub/mozilla.org/xulrunner/releases/

Then copy from /Library/Frameworks/XUL.framework/Versions/1.9/... make sure you copy all symlinks correctly (use rsync -rl /Library/Frameworks/XUL.framework ...).


more information of deploying XUL apps is here:
